Closed Caption Disassembly Documentation: Codes

The following table lists all of the codes that are not on the characters page. The last columns of each row specify the hexidecimal equivalents for Channels 1 through 4 in the SCC file.

Note: The special code {??} is used by CCASDI whenever it does not understand a code in SCC format. The program will refuse to reverse the translation until all {??} codes are replaced with valid codes or characters.

Note that Text channels are completely separate from the Caption channels--you can have Text and Captions in use simultaneously and switch which one is visible using your TV remote control.
